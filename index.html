RecGym Dataset Introduction

The RecGym dataset is a collection of 10 gym exercises with 615 abstracted features, designed for research and development in recommendation systems and fitness applications.

Dataset Overview

  • Number of Exercises: 10
  • Number of Features: 615
  • Purpose: To provide a robust dataset for building and testing recommendation systems in fitness contexts.

Features

The dataset includes abstracted features such as:

  • Exercise type
  • Duration
  • Intensity level
  • User preferences
  • And more...
